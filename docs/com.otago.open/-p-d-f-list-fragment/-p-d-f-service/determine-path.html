determinePath

fun determinePath(currentDir: String, href: String, paperUrl: String): String?

Determines where a href on a page given by currentDir should point, if currentDir is a file (i.e. no forward slash), otherwise when in the folder given by currentDir

If href starts with a "/" We just return the href with the paperUrl removed

The return is relative to paperUrl

Parameters

currentDir - The current folder or page

href - The href

paperUrl - The base url for the course

Return
Where href should point (relative to paperUrl) if it appears at currentDir
